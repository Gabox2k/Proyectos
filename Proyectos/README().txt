Requisitos
-Guardar el codigo en un archivo .c
-Se usa librería stdio.h (para printf)
-librería stdlib.h (para rand, srand)
-librería time.h (para time)

Proyecto 
En este proyecto se genera un laberinto en c usando backtrackin y
encuentra el camino usando BFS

Juego
-Es un tablero de 15x15
-Genera de forma aleatoria el laberinto
-Marca el camino correcto (+) desde E(Entrada) hasta S(salida)

Objetivo
Es generar automaticamente un laberinto aleatorio y luego buscar el camino mas corto desde un piunto de incio hasta una salida
usando el algoritmo BFS. El camino correcto se marcara con (+)
