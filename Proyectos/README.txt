Requisitos
-Se usa la aplicacion visual studio (python 3.x)
-Guardar el código en un archivo .py
-Se usa la librería random (import random)

Proyecto 
En este proyecto es un juego que el ratón tiene que comer todo el queso para ganar el gato y el gato tiene
que atrapar al ratón antes que coma todo el queso

Dificultad 
Mi mayor dificultad fue la implementacion del minimax para el gato

Logro
Que aparezcan los quesos y los obstaculos en forma aleatoria 

Juego
-Es un tablero de 10x10
-El ratón empieza en una posición  fija 
-El gato empieza en una posición fija
-El queso aparece de forma aleatoria
-El obstáculo aparece de forma aleatoria

Objetivo
-Comer todos los quesos antes que te atrape el gato
-Comer mas de 4 quesos para desbloquear otros movimientos
-Ganas cuando comes todo el queso
-Perdes cuando el gato te atrape
